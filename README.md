# 422_Tsyguleva_Pushkina

Этот проект представляет собой WPF-приложение на C# с использованием XAML для интерфейса. Основная функциональность связана с управлением пользователями, категориями, платежами и аутентификацией.

## Структура проекта

### Pages

* AddCategoryPage.xaml – Страница для добавления новых категорий.
* AddPaymentPage.xaml – Страница для добавления информации о платежах.
* AddUserPage.xaml – Страница для добавления новых пользователей.
* AdminPage.xaml – Страница с административными функциями.
* AuthPage.xaml – Страница аутентификации пользователей.
* CategoryTabPage.xaml – Страница с отображением категорий в виде вкладок.
* ChangePassPage.xaml – Страница для изменения пароля текущего пользователя.
* DiagrammPage.xaml & DiagrammPage.xaml.cs – Страница для отображения диаграмм и графиков.
* PaymentTabPage.xaml – Страница с отображением платежей в виде вкладок.
* RegPage.xaml – Страница регистрации новых пользователей.
* UserPage.xaml – Страница управления данными конкретного пользователя.
* UsersTabPage.xaml – Страница с отображением списка пользователей в виде вкладок.

### Основные файлы

* App.xaml / App.config – Конфигурация приложения.
* MainWindow.xaml – Главное окно приложения.
* DbContextHelper.cs – Класс для работы с базой данных.
* Model1.edmx – EDMX модель для работы с базой данных.
* Dictionary.xaml / Dictionary1.xaml – Ресурсы и словари стилей.
* packages.config – Управление пакетами NuGet.

## Возможности

* Управление пользователями (добавление, редактирование, просмотр).
* Управление категориями и платежами.
* Аутентификация и регистрация пользователей.
* Административная панель.
* Визуализация данных в виде диаграмм и таблиц.

## Инструкция по запуску

1. Открыть проект в Visual Studio.
2. Настроить подключение к базе данных через DbContextHelper.cs и Model1.edmx.
3. Построить и запустить проект.
4. Использовать соответствующие страницы для управления пользователями, категориями и платежами.
